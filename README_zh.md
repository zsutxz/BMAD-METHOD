## 什么是 BMad-CORE？

### v6 核心增强

- **🎨 代理定制** - 通过 `bmad/_cfg/agents/` 修改名称、角色、个性
- **🌐 多语言** - 独立的通信和输出语言设置
- **👤 个性化** - 代理适应您的姓名、技能水平和偏好
- **🔄 持久配置** - 自定义在模块更新后保留
- **⚙️ 灵活设置** - 按模块或全局配置

### C.O.R.E. 哲学

- **C** 协作（Collaboration）：人机伙伴关系，利用互补优势
- **O** 优化（Optimized）：经过实战检验的流程，实现最大效果
- **R** 反思（Reflection）：战略性提问，解锁突破性解决方案
- **E** 引擎（Engine）：协调 19+ 个专业化代理和 50+ 个工作流程的框架

BMad-CORE 不会给您答案——它通过指导反思帮助您**发现更好的解决方案**。

## 🎯 使用代理和命令

### 方法 1：代理菜单（推荐给初学者）

加载任何代理并运行交互式菜单：
```bash
*menu
```

### 方法 2：直接斜杠命令

直接执行命令：
```bash
/agent-name command-name
```

### 方法 3：派对模式执行

使用 * 前缀快速执行：
```bash
*agent-command
```
